> ### 전시회가 나에게 다가왔다...🥺 !
> 기존 전시회 정보들이 여러 커뮤니티 사이트에 산재되어 있던 문제들을
하나의 커뮤니티에서 전시회의 정보를 한눈에 볼 수 있는 서비스를 개발함 으로써
모든 전시회 관람객 및 관계자들의 불편을 해결하려합니다 !
